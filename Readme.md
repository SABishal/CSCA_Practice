# CSCA Practice

An online practice platform for the China Scholastic Competency Assessment (CSCA) exam, built with vanilla JavaScript, Vercel Serverless Functions, and Cloudflare.

## Live Demo
[https://csca-test.org](https://csca-test.org)

## Features
- Chapter‑wise and mock tests for Mathematics, Physics, Chemistry
- Smart question selection to avoid repetition
- Timed quizzes with instant feedback
- Progress tracking with localStorage
- Optimized access from China via Cloudflare

## Tech Stack
HTML, CSS, JavaScript, Vercel, Cloudflare, EmailJS

## Local Development
Open `index.html` with a local server (e.g., VS Code Live Server).
*Note: The `/api/generate` endpoint works only when deployed on Vercel.*
